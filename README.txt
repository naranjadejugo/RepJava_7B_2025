# Ejercicios en Java

Este repositorio contiene 22 ejercicios básicos de programación en Java con entrada de datos por teclado.

## Requisitos

- Tener instalado JDK (Java Development Kit)
- Usar un editor de texto o IDE como Visual Studio Code, IntelliJ IDEA o NetBeans
- Asegurarte de tener `javac` y `java` configurados en la variable de entorno PATH

## Compilar un ejercicio

Desde la terminal o consola:

```bash
javac Ejercicio1.java
```

Esto generará un archivo `Ejercicio1.class`.

## Ejecutar un ejercicio

Luego de compilar, ejecutá:

```bash
java Ejercicio1
```

Repetí el proceso para cualquiera de los archivos `EjercicioX.java`, donde X es el número del ejercicio.

## Ejemplos

```bash
javac Ejercicio5.java
java Ejercicio5
```

## Notas

- Si usás Windows, podés abrir la terminal `cmd` o PowerShell.
- Si estás en Linux o Mac, usá la terminal común.
- Todos los ejercicios usan `Scanner` para leer entrada desde teclado.

¡Buena práctica!
